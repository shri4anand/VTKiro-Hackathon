# VTKiro-Hackathon
AI Crisis Text Simplifier (Accessibility Focus)

Problem: Emergency alerts and crisis info are often too complex for people with low literacy, non-native speakers, or cognitive disabilities.
Solution: Paste any alert → AI rewrites it into:

Plain language
Multiple reading levels
Multiple languages
Audio playback

Why it scores high:

Social Good (12/12): Accessibility + emergency response
AI Integration: Prompt engineering + readability transformation + summarization
Impact: Easy to demo with real alerts

Tech Stack:

Frontend: React + Tailwind
Backend: Node/Python
AI: LLM (prompt templates for grade-level rewriting)
Optional: Text-to-speech API

10-hour scope:

Input box → output simplified versions + language toggle
